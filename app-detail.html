数のお勉強アプリ

2025年1月26日
教育
子ども向けの数の概念を学ぶ教育アプリです。親が設定した数だけアイテムを表示し、子どもが数を数える練習ができます。カウンティングゲーム機能で楽しく学習でき、進捗を記録することも可能です。
アプリを試してみる ソースコードを見る

主な機能

  • 数の表示（1-10）
  • 8種類のアイテム選択
  • カウンティングゲーム
  • 学習記録の保存
  • 進捗統計の表示
  • 親のフィードバック機能

使用技術

HTML5 CSS3 JavaScript LocalStorage

学習ポイント

  • DOM操作とイベント処理
  • LocalStorageでのデータ永続化
  • CSS アニメーション
  • レスポンシブデザイン
  • UX/UI設計
